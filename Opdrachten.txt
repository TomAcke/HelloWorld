hierboven vermelde stappen. Kijk telkens na of je die files terugvindt op GitHub. 
Wijzig nu één van de files en zorg dat de wijziging doorgevoerd wordt naar GitHub.

tutorial over Git. Neem deze tutorial zoveel mogelijk door (zowel main als remote)

willekeurige README-file. Maak een nieuwe file aan: RubyProg1.rb met 
onderstaande code. Probeer de code te begrijpen en zet deze file ook op GitHub.
